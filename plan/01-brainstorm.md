# 01 - Brainstorm

## Role
AI berperan sebagai UX/UI consultant untuk portfolio developer.

## Task
Membantu brainstorm desain portfolio untuk fresh graduate IT yang ingin menampilkan profil, skill, project, dan kontak secara profesional.

## Context
- Nama: Zhafran Hafizh I.R
- Role yang dituju: Software Engineer
- Skills: HTML, CSS, JavaScript, React, Next.js, Tailwind CSS, Git, GitHub
- Target: Startup dan corporate recruiter
- Tujuan portfolio: Menjadi personal website sederhana yang bisa digunakan untuk melamar kerja, menampilkan project, dan membangun personal branding.

## Constraints
- Mobile-first
- Simple tetapi tetap stand out
- Tidak terlihat terlalu ramai
- Mudah dibaca oleh recruiter dalam waktu singkat
- Menggunakan HTML dan CSS tanpa framework

## Output

### 1. Color Palette Options

#### Option 1 — Modern Dark
- Background: #0F172A
- Surface/Card: #111827
- Primary Text: #F8FAFC
- Secondary Text: #CBD5E1
- Accent: #38BDF8

Alasan: Cocok untuk developer portfolio karena terlihat modern, clean, dan profesional.

#### Option 2 — Clean Light
- Background: #F8FAFC
- Surface/Card: #FFFFFF
- Primary Text: #0F172A
- Secondary Text: #475569
- Accent: #2563EB

Alasan: Lebih aman untuk target corporate karena tampilannya bersih, mudah dibaca, dan profesional.

#### Option 3 — Soft Minimal
- Background: #FAFAF9
- Surface/Card: #FFFFFF
- Primary Text: #1C1917
- Secondary Text: #57534E
- Accent: #EA580C

Alasan: Terlihat lebih hangat dan personal, cocok untuk membuat portfolio terasa lebih human dan tidak terlalu kaku.

### 2. Typography Recommendation

Font recommendation:
- Primary font: Inter, system-ui, sans-serif
- Heading: Bold, clean, dan memiliki hierarchy jelas
- Body text: Regular dengan line-height yang nyaman dibaca

Alasan:
Inter atau system font cocok untuk website portfolio karena tampil modern, ringan, dan mudah dibaca di berbagai device.

### 3. Section Structure

Portfolio akan menggunakan struktur one-page sebagai berikut:

1. Header / Navigation
   - Nama atau logo sederhana
   - Menu navigasi ke About, Skills, Projects, Contact

2. Hero Section
   - Perkenalan singkat
   - Role/title
   - CTA ke project dan contact

3. About Section
   - Ringkasan singkat tentang diri
   - Fokus minat di bidang IT/web development

4. Skills Section
   - Daftar skill teknis
   - Ditampilkan dalam bentuk badge atau card kecil

5. Projects Section
   - Beberapa project utama
   - Setiap project berisi nama, deskripsi singkat, tech stack, dan link

6. Contact Section
   - Email
   - LinkedIn
   - GitHub

7. Footer
   - Copyright sederhana

### 4. Unique Element

Unique element yang dipilih:
"Project highlight card" dengan label seperti Featured Project.

Alasan:
Recruiter biasanya ingin melihat hasil kerja dengan cepat. Dengan adanya featured project, project terbaik bisa langsung terlihat tanpa harus membaca seluruh halaman terlalu lama.

## Final Design Direction

Saya memilih pendekatan modern minimalis dengan layout one-page, mobile-first, dan fokus pada keterbacaan. Desain tidak dibuat terlalu ramai agar informasi utama seperti profil, skill, project, dan kontak bisa dipahami dengan cepat oleh recruiter.
